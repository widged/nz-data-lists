New Zealand APIs

Skip to end of metadata
  • Created by Unknown User (wiki-import) on 08 Nov, 2011
Go to start of metadata

This list of NZ APIs was pulled together for some research, and others may find it useful. Some of these are from commercial providers (hope it's okay to list them here), but they are mostly free. This is just what could be easily found. If you know of others please add, and perhaps we can get them added to the various data catelogues if they're not already included.

GeoNet Earthquake data

http://www.geonet.org.nz/resources/earthquake/quake-web-services.html

TradeMe

http://developer.trademe.co.nz/

NZ event info

http://www.eventfinder.co.nz/api/index

Metadata & thumbnails of NZ content objects

http://www.digitalnz.org/developer

NZ highway info

https://infoconnect.highwayinfo.govt.nz/opencms/opencms/infoconnect

Koordinates geospatial data

http://api.koordinates.com

Send and receive SMS via NZ API gateway

http://www.message-media.co.nz/sms-gateway.html

Reusable tourism web content

http://www.newzealand.com/travel/utilities/help/web-services.cfm

Wellington news

http://wellington.gen.nz/api

Integrate with Xero services

https://network.xero.com/Login.xro/Login

NZ vehicle information

http://www.carjam.co.nz/cms/2008/12/24/carjam-api/

Track NZ post deliveries

http://www.nzpost.co.nz/Cultures/en-NZ/ProductsAndServices/D/DeveloperCentre/TrackingAPI/TrackingAPIOverview.htm

NZ weather (international site)

http://wiki.wunderground.com/index.php/API_-_XML

NZ climate data (not really API but can use curl)

http://cliflo.niwa.co.nz/

Zenubu listings about NZ places

http://www.zenbu.co.nz/about/xml

Availability of .nz domains

You can use the official 'whois' service by (1) open a TCP connection to whois.srs.net.nz on port 43 (2) send the domain name followed by a newline character. The service will respond with information about the domain name, or that it is not registered.
You can also use a (wide) variety of online services such as the one described here: http://www.simplename.co.nz/page/api

Latest 50 Trademe auctions

http://archive.votemenot.co.nz/thread/25651432/trademe-api/ http://rowansimpson.com/2007/05/15/a-conversation-about-an-api/

Personal bankruptcies and companies in liquidation

http://www.insolvency.govt.nz/cms/customer-support/news-and-public-notices/important-announcements/web-service-now-available-for-insolvency-and-trustee-service/view

Companies office listings

http://www.business.govt.nz/companies/help-support/technical-support/connect-direct/web-services

RSS feed of re-published historical NZ books

http://www.nzetc.org/tm/scholarly/facets/rss?fq=record_type:facet&fq=record_type:facet&fq=type:Work&facet=true&style=rss&rows=5000

Powershop (monitor usage, enter meter readings and top up your account)

http://powershop.co.nz/developers.html
